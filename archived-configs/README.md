# Archived Configuration Files

This directory contains legacy configuration files that are no longer in use but are kept for historical reference.

## Render Configuration Files

The following files are from the original **Render** deployment approach, which has been replaced with **Azure VM** deployment:

- `render.yaml` - Render deployment configuration
- `RENDER_CHECKLIST.md` - Render deployment checklist
- `RENDER_DEPLOYMENT.md` - Render deployment guide
- `RENDER_DEPLOYMENT_GUIDE.md` - Detailed Render deployment instructions
- `RENDER_FORM_COPY_PASTE.sh` - Shell script for Render form automation
- `RENDER_DOMAIN_FIX.txt` - Render domain configuration fix notes
- `READY_FOR_RENDER.md` - Render readiness documentation

## Why These Files Are Archived

The project has migrated from **Render** (a cloud-based deployment platform) to **Azure VM** (virtual machine-based deployment). The Azure approach provides:

- Greater control over infrastructure
- Better performance for the application's requirements
- Cost optimization through dedicated resources
- Custom deployment configurations

These Render files are kept in this archive directory for historical reference and to understand the project's evolution. They should not be used for current deployments.
