1. Deploy Backend (2 min)
   ─────────────────────
   Go to: https://render.com
   Sign up → New Web Service
   Select: megaverselive repository
   Root Directory: backend
   Start Command: npm start
   
   Add Environment Variables (from backend/.env)
   Click Deploy
   
   Get your URL: https://megaverse-api.onrender.com
   
2. Update GoDaddy DNS (5 min)
   ───────────────────────────
   Go to: godaddy.com → Your Domain → Manage DNS
   Add CNAME:
     Name: api
     Value: megaverse-api.onrender.com
     TTL: 3600
   
   Test: curl https://api.megaverselive.com/api/health
   
3. Update index.html (5 min)
   ──────────────────────────
   Edit: index.html
   Replace API_URL with: https://api.megaverselive.com
   Add Stripe Publishable Key
   Add booking form (see FRONTEND_INTEGRATION.md)
   
4. Test & Done!
   ─────────────
   Test booking with Stripe test card: 4242 4242 4242 4242
   Check email for confirmation
