NEW API ENDPOINTS:
==================

1. Razorpay Webhook
   POST /api/razorpay/webhook
   - Receives payment notifications from Razorpay
   - Verifies with signature
   - Stores in database

2. PayPal Webhook
   POST /api/paypal/webhook
   - Receives payment notifications from PayPal
   - Stores in database

3. Check Payment Status
   GET /api/payment/:paymentId
   - Returns payment status and metadata
   - Use to verify if payment succeeded

NEW DATABASE TABLE:
===================

payment_records:
- payment_id: Unique ID from provider
- provider: razorpay or paypal
- amount: Payment amount
- currency: INR or USD
- status: completed or failed
- metadata: Full payment data (for disputes)
- Indexes for fast lookups

NEXT STEPS (for you):
====================

1. Configure Razorpay Webhook:
   - Go to Razorpay Dashboard
   - Settings → Webhooks
   - Add: https://megaverselive.com/api/razorpay/webhook
   - Select events: payment.authorized, payment.failed
   - Add RAZORPAY_WEBHOOK_SECRET to Render environment

2. Configure PayPal Webhook (optional):
   - Go to PayPal Developer Dashboard
   - Webhooks
   - Add: https://megaverselive.com/api/paypal/webhook
   - Select: CHECKOUT.ORDER.COMPLETED

3. Test:
   - Make a test payment
   - Check database: SELECT * FROM payment_records
   - Should see your payment with status='completed'
